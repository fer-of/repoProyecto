6. Software


Descripcion del Software.

Este software permite la visualización de algoritmos de búsqueda en grafos, específicamente BFS (Breadth-First Search) y DFS (Depth-First Search).
Los usuarios pueden seleccionar un nodo de inicio y ejecutar cualquiera de los algoritmos para ver cómo se exploran los nodos del grafo.

Instrucciones para el Usuario

Seleccionar el Nodo de Inicio:

Utiliza el menú desplegable para seleccionar el nodo desde el cual deseas iniciar la búsqueda.
Ejecutar un Algoritmo:

Haz clic en el botón "Ejecutar BFS" para ver cómo el algoritmo BFS explora los nodos del grafo.
Haz clic en el botón "Ejecutar DFS" para ver cómo el algoritmo DFS explora los nodos del grafo.
Visualización del Grafo:

Observa el contenedor del grafo (graph-container) para ver la visualización de la exploración de los nodos según el algoritmo seleccionado.
